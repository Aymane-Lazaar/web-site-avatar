# Avatar Web Page

This is a simple HTML and CSS project for a landing page called "Avatar". The page features a modern gradient background, a navigation menu, and a hero section with an image and button.

## Project Structure


- `index.html` → The main HTML file containing the structure of the page.
- `style web avatar.css` → CSS file for styling the page.
- `avatar-img/avatar.webp` → Background image used in the hero section.

## Features

- Responsive container with centered content.
- Gradient overlay on background image.
- Hover effect on container to make the overlay transparent.
- Navigation bar with links: Home, Class, Videos, Photos, About.
- Hero section with a heading, paragraph, and a styled button.
- Modern typography with sans-serif font and italic style.
